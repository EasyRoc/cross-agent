## 1. 报告增加图片信息

报告图片分为两类：**数据可视化图表**（分析结果的统计展示）和 **商品实物图片**（真实商品外观展示）。

---

### 1.1 数据可视化图表

#### 现状
当前分析报告仅包含文本内容，缺少可视化图表，影响数据呈现效果和报告可读性。

#### 优化目标
在报告中自动生成数据可视化图片（饼图、柱状图、趋势图等），嵌入 Markdown/PDF 报告中。

#### 实现方案

```
分析数据 → matplotlib/plotly 生成图表 → 保存为图片
    → Markdown: 通过 ![alt](path) 引用本地图片
    → PDF: 将 base64 图片嵌入 HTML
```

**具体步骤**：

1. **图表生成服务**（新增 `app/report/charts.py`）：
   - 价格带分布图（柱状图）
   - 配色/材质/风格占比（饼图）
   - 销量 TOP 排行（横向柱状图）
   - 生命周期分布（堆叠柱状图）
   - 人群画像雷达图

2. **嵌入报告**：
   - 分析完成后，ReportGenerator 调用图表服务生成图片
   - Markdown 格式使用相对路径引用图片文件
   - PDF 格式将图片转为 base64 嵌入 HTML

3. **依赖新增**：
   ```toml
   "matplotlib>=3.8.0",
   "plotly>=5.18.0",
   "pillow>=10.0.0",
   ```

4. **图表存储**：
   - 图片文件存放于 `reports/charts/{session_id}/` 目录
   - 报告生成后定期清理临时图片

### 1.2 商品实物图片

#### 现状
分析报告中仅有文字描述，没有真实的商品外观展示图，用户无法直观了解商品外观、配色、设计风格。

#### 优化目标
在报告中自动嵌入商品实物图片，让用户在看分析数据时能直接看到商品长什么样，提升报告的直观性和说服力。

2. **新增 `app/report/image_fetcher.py`**：下载和缓存图片

```python
import aiohttp
import aiofiles
from pathlib import Path

class ImageFetcher:
    """商品图片下载与缓存"""

    def __init__(self, cache_dir="reports/images"):
        self.cache_dir = Path(cache_dir)
        self.cache_dir.mkdir(parents=True, exist_ok=True)

    async def download(self, url: str, filename: str) -> str:
        """下载图片到本地缓存，返回本地路径"""
        ext = url.rsplit('.', 1)[-1].split('?')[0] or 'jpg'
        filepath = self.cache_dir / f"{filename}.{ext}"
        if filepath.exists():
            return str(filepath)
        async with aiohttp.ClientSession() as session:
            async with session.get(url, timeout=10) as resp:
                if resp.status == 200:
                    async with aiofiles.open(filepath, 'wb') as f:
                        await f.write(await resp.read())
        return str(filepath)

    async def batch_download(self, products: list[dict]) -> list[str]:
        """批量下载商品图片"""
        import asyncio
        tasks = []
        for i, p in enumerate(products):
            images = p.get("tags", [])  # 从 tags 取图片 URL
            if images and images[0].startswith("http"):
                tasks.append(self.download(images[0], f"product_{i}"))
        return await asyncio.gather(*tasks, return_exceptions=True)
```

## 2. OSS 上传最终生成文件

### 现状
报告文件仅保存在本地 `reports/` 目录，无法持久化存储和跨实例共享。

### 优化目标
报告生成后自动上传到阿里云 OSS，返回可公开访问的下载链接。

### 实现方案

**新增 OSS 上传模块**（`app/storage/oss.py`）：

```python
import oss2
from app.config import settings

class OSSStorage:
    def __init__(self):
        self.auth = oss2.Auth(
            settings.OSS_ACCESS_KEY_ID,
            settings.OSS_ACCESS_KEY_SECRET
        )
        self.bucket = oss2.Bucket(
            self.auth,
            settings.OSS_ENDPOINT,
            settings.OSS_BUCKET_NAME
        )

    async def upload_file(self, local_path: str, remote_path: str) -> str:
        """上传文件到 OSS，返回公开访问 URL"""
        self.bucket.put_object_from_file(remote_path, local_path)
        return f"https://{settings.OSS_BUCKET_NAME}.{settings.OSS_ENDPOINT}/{remote_path}"
```

```

**流程变更**：

```

## 3. 长期记忆

### 现状
当前仅实现了短期记忆（最近 20 轮对话）和摘要记忆（超出窗口后总结），重启会话后所有记忆丢失。

### 优化目标
实现跨会话的长期记忆，持久化存储用户的历史分析记录，包括：
- 用户偏好（常用平台、关注维度、偏好的报告格式）
- 历史分析记录（曾分析过的商品、生成的报告）
- 历史反馈（用户曾提出过的改进建议）

### 实现方案

#### 3.1 记忆存储

**方案 A：SQLite（轻量，推荐初期使用）**

```python
# app/memory/long_term.py
import sqlite3
import json
from pathlib import Path

class LongTermMemory:
    """长期记忆（跨会话持久化）"""

    def __init__(self, db_path: str = "data/memory.db"):
        self.db_path = Path(db_path)
        self.db_path.parent.mkdir(parents=True, exist_ok=True)
        self._init_db()

    def _init_db(self):
        conn = sqlite3.connect(str(self.db_path))
        conn.execute("""
            CREATE TABLE IF NOT EXISTS user_memory (
                session_id TEXT PRIMARY KEY,
                user_preferences TEXT,  -- JSON: 偏好设置
                analysis_history TEXT,  -- JSON: 历史分析记录
                feedback_history TEXT,  -- JSON: 历史反馈
                created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
            )
        """)
        conn.commit()
        conn.close()

    def save_preference(self, session_id: str, key: str, value: any):
        """保存用户偏好"""
        prefs = self.get_preferences(session_id)
        prefs[key] = value
        conn = sqlite3.connect(str(self.db_path))
        conn.execute(
            """INSERT INTO user_memory (session_id, user_preferences)
               VALUES (?, ?) ON CONFLICT(session_id)
               DO UPDATE SET user_preferences = ?, updated_at = CURRENT_TIMESTAMP""",
            (session_id, json.dumps(prefs), json.dumps(prefs))
        )
        conn.commit()
        conn.close()
```

- 将记忆也向量化存入 Milvus，与商品检索共用向量库

#### 3.2 记忆内容

## 5. 自定义分析 SKILL

用户可以定义可复用的分析 SKILL（技能模板），每个 SKILL 封装了特定的分析逻辑、平台偏好、维度选择和 Prompt 指令。在对话中通过名称引用 SKILL，实现"一键分析"。

### 5.1 什么是 SKILL

]


class SkillStore:
    """SKILL 持久化存储"""

    def __init__(self, db_path: str = "data/memory.db"):
        self.db_path = Path(db_path)
        self.db_path.parent.mkdir(parents=True, exist_ok=True)
        self._init_db()

    def _init_db(self):
        conn = sqlite3.connect(str(self.db_path))
        conn.execute(SKILL_INIT_SQL)
        # 初始化预置 SKILL
        for skill in PRESET_SKILLS:
            conn.execute(
                "INSERT OR IGNORE INTO skills (id, name, definition) VALUES (?, ?, ?)",
                (skill.id, skill.name, skill.model_dump_json())
            )
        conn.commit()
        conn.close()

    def list_all(self) -> list[Skill]:
        """列出所有 SKILL"""
        conn = sqlite3.connect(str(self.db_path))
        rows = conn.execute(
            "SELECT definition FROM skills ORDER BY use_count DESC"
        ).fetchall()
        conn.close()
        return [Skill.model_validate_json(r[0]) for r in rows]

    def get_by_name(self, name: str) -> Optional[Skill]:
        """按名称查找 SKILL"""
        conn = sqlite3.connect(str(self.db_path))
        row = conn.execute(
            "SELECT definition FROM skills WHERE name = ?", (name,)
        ).fetchone()
        conn.close()
        if row:
            return Skill.model_validate_json(row[0])
        return None

    def save(self, skill: Skill):
        """保存或更新 SKILL"""
        conn = sqlite3.connect(str(self.db_path))
        skill.updated_at = datetime.now()
        conn.execute(
            """INSERT INTO skills (id, name, definition, updated_at)
               VALUES (?, ?, ?, CURRENT_TIMESTAMP)
               ON CONFLICT(id) DO UPDATE SET
               definition=?, name=?, updated_at=CURRENT_TIMESTAMP""",
            (skill.id, skill.name, skill.model_dump_json(),
             skill.model_dump_json(), skill.name)
        )
        conn.commit()
        conn.close()

    def increment_use(self, skill_id: str):
        """增加使用计数"""
        conn = sqlite3.connect(str(self.db_path))
        conn.execute("UPDATE skills SET use_count = use_count + 1 WHERE id = ?", (skill_id,))
        conn.commit()
        conn.close()

    def delete(self, skill_id: str):
        """删除 SKILL"""
        conn = sqlite3.connect(str(self.db_path))
        conn.execute("DELETE FROM skills WHERE id = ?", (skill_id,))
        conn.commit()
        conn.close()

```python
# app/agent/nodes/skill_matcher.py

import re
from app.memory.skill_store import SkillStore
from app.logger import get_logger

logger = get_logger(__name__)

# 匹配模式：用 + 技能名 + 分析 + 商品
SKILL_PATTERNS = [
    r"(?:用|使用|调用)(.+?)(?:分析|查看|做)",
    r"(?:用|使用|调用)(.+?)技能",
    r"(.+?)分析(?:一下)?$",  # 后置匹配：爆款挖掘分析男式衬衫
]

async def match_skill(user_input: str) -> tuple[Optional[dict], str]:
    """
    从用户输入中匹配 SKILL。

    Args:
        user_input: 用户输入文本

    Returns:
        (skill_dict, clean_query) 匹配到 SKILL 则返回技能定义和去除技能名后的查询，
        否则返回 (None, 原始输入)
    """
    store = SkillStore()
    skills = store.list_all()

    for pattern in SKILL_PATTERNS:
        match = re.search(pattern, user_input)
        if match:
            skill_name = match.group(1).strip()
            # 尝试匹配已知 SKILL
            for skill in skills:
                if skill.name in skill_name or skill_name in skill.name:
                    logger.info("匹配到 SKILL: %s", skill.name)
                    store.increment_use(skill.id)
                    # 从输入中去除 SKILL 名，保留商品查询
                    clean = user_input.replace(match.group(0), "", 1).strip()
                    return skill.model_dump(), clean or skill_name

    return None, user_input
```

from datetime import datetime, timedelta
from uuid import uuid4

JWT_SECRET_KEY = "change-me-in-production"  # 后续移入 .env

JWT_EXPIRE_HOURS = 24

def hash_password(password: str) -> str:
    return bcrypt.hashpw(password.encode(), bcrypt.gensalt()).decode()

def verify_password(password: str, hashed: str) -> bool:
    return bcrypt.checkpw(password.encode(), hashed.encode())

def create_token(user_id: str) -> str:
    payload = {
        "sub": user_id,
        "jti": str(uuid4()),
        "exp": datetime.utcnow() + timedelta(hours=JWT_EXPIRE_HOURS),
    }
    return jwt.encode(payload, JWT_SECRET_KEY, algorithm=JWT_ALGORITHM)

```python
import sqlite3
from pathlib import Path
from datetime import datetime
from uuid import uuid4

class TaskManager:
    """异步任务管理器"""

    STATUS_PENDING = "pending"
    STATUS_RUNNING = "running"
    STATUS_COMPLETED = "completed"
    STATUS_FAILED = "failed"

    def __init__(self, db_path: str = "data/tasks.db"):
        self.db_path = Path(db_path)
        self.db_path.parent.mkdir(parents=True, exist_ok=True)
        self._init_db()

    def _init_db(self):
        conn = sqlite3.connect(str(self.db_path))
        conn.execute("""
            CREATE TABLE IF NOT EXISTS tasks (
                id TEXT PRIMARY KEY,
                user_id TEXT NOT NULL,
                product_name TEXT NOT NULL,
                status TEXT DEFAULT 'pending',
                progress TEXT DEFAULT '',          -- JSON: 各步骤进度
                report_md_path TEXT DEFAULT '',    -- Markdown 文件路径
                report_pdf_path TEXT DEFAULT '',   -- PDF 文件路径
                error_message TEXT DEFAULT '',
                created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
                FOREIGN KEY (user_id) REFERENCES users(id)
            )
        """)
        conn.execute("""
            CREATE INDEX IF NOT EXISTS idx_tasks_user
            ON tasks(user_id, created_at DESC)
        """)
        conn.commit()
        conn.close()

    def has_running_task(self, user_id: str) -> bool:
        """检查用户是否有运行中的任务"""
        conn = sqlite3.connect(str(self.db_path))
        row = conn.execute(
            "SELECT id FROM tasks WHERE user_id=? AND status IN ('pending','running')",
            (user_id,)
        ).fetchone()
        conn.close()
        return row is not None

    def create_task(self, user_id: str, product_name: str) -> str:
        """创建新任务，如果用户已有活跃任务则抛出异常"""
        if self.has_running_task(user_id):
            raise RuntimeError("用户已有进行中的任务，请等待完成")
        task_id = str(uuid4())
        conn = sqlite3.connect(str(self.db_path))
        conn.execute(
            "INSERT INTO tasks (id, user_id, product_name) VALUES (?, ?, ?)",
            (task_id, user_id, product_name)
        )
        conn.commit()
        conn.close()
        return task_id

    def update_task(self, task_id: str, **kwargs):
        """更新任务字段"""
        fields = ", ".join(f"{k}=?" for k in kwargs)
        values = list(kwargs.values()) + [task_id]
        conn = sqlite3.connect(str(self.db_path))
        conn.execute(
            f"UPDATE tasks SET {fields}, updated_at=CURRENT_TIMESTAMP WHERE id=?",
            values
        )
        conn.commit()
        conn.close()

    def get_task(self, task_id: str) -> dict:
        conn = sqlite3.connect(str(self.db_path))
        conn.row_factory = sqlite3.Row
        row = conn.execute("SELECT * FROM tasks WHERE id=?", (task_id,)).fetchone()
        conn.close()
        return dict(row) if row else None

    def list_user_tasks(self, user_id: str, limit: int = 20, offset: int = 0) -> list[dict]:
        conn = sqlite3.connect(str(self.db_path))
        conn.row_factory = sqlite3.Row
        rows = conn.execute(
            "SELECT * FROM tasks WHERE user_id=? ORDER BY created_at DESC LIMIT ? OFFSET ?",
            (user_id, limit, offset)
        ).fetchall()
        conn.close()
        return [dict(r) for r in rows]
```

import asyncio
from app.task.manager import TaskManager
from app.agent.graph import create_analysis_graph
from app.logger import get_logger

logger = get_logger(__name__)


def run_analysis_task(task_id: str, user_id: str, params: dict):
    """在后台线程中执行分析任务"""
    loop = asyncio.new_event_loop()
    asyncio.set_event_loop(loop)
    try:
        manager = TaskManager()
        manager.update_task(task_id, status="running", progress='{"step": "数据采集"}')

        # 构建并执行 LangGraph
        graph = create_analysis_graph()
        initial_state = {
            "params": params,
            "messages": [],
            "session_id": f"task_{task_id}",
        }
        result = loop.run_until_complete(graph.ainvoke(initial_state))

        # 保存报告路径
        report_paths = result.get("report_paths", {})
        manager.update_task(
            task_id,
            status="completed",
            progress='{"step": "completed"}',
            report_md_path=report_paths.get("markdown", ""),
            report_pdf_path=report_paths.get("pdf", ""),
        )
        logger.info("任务 %s 完成", task_id)
    except Exception as e:
        logger.error("任务 %s 失败: %s", task_id, e, exc_info=True)
        manager.update_task(task_id, status="failed", error_message=str(e))
    finally:
        loop.close()

### 6.9 风险

| 风险 | 说明 | 缓解措施 |
|------|------|----------|
| WebSocket 断连重连 | 用户刷新后需重新鉴权 | token 存 localStorage，重连时自动携带 |
| 异步任务并发 | 后台任务可能堆积 | 每个用户限制 1 个活跃任务 |
| 文件隔离 | 用户 A 不可下载用户 B 的报告 | 目录按 user_id 隔离，API 校验归属 |
| 密码安全 | 密码泄露 | bcrypt + HTTPS（生产环境） |
